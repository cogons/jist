# Jist

> A Code Snippets Manager for Github Gist.

![](preview.jpeg)

#### Feature

- Local storage
- Quick search
- More meta information
- Code highlighting
- Sync with Github Gist

#### Build Setup

``` bash
# install dependencies
yarn install

# run development
yarn dev

# build electron application for production
yarn build

```
